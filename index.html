jQuery Tutorialmaker

This is a simple jQuery plugin to make a tutorial or a "Take a tour"-Feature on your web application.

Take a tour

A step of your tutorial

You need to define the steps of your tutorial. A single step is build like this:

{
  step: 'a name for the step', // not displayed
  elem: $('#element-to-display'),
  position: 'bottom', // position of the helpbox: left | right | top | bottom
  button: 'Next', // button text: null or text
  title: 'Title',
  body: 'Your help text.',
}

Init your tutorialmaker

First of all, init your tutorialmaker:

var helper = $.tm({
  steps: []
});

Move to next step

Move to the next step with executing this code:

helper.next();

For example you can give your user a task to do, before he can move on.

Let's go

Click on the button to go to the next step.
