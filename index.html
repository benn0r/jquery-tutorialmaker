jQuery Tutorialmaker

This is a simple jQuery plugin to make a tutorial or a "Take a tour"-Feature on your web application.

Take a tour

Step 1

Init your tutorialmaker. You need to define the steps of your tutorial. A single step is build like this:

{
  step: 'a name for the step',
  elem: $('#step0'),
  position: 'bottom',
  button: 'Next',
  title: 'Step 0',
  body: 'Vestibulum erat wisi, condimentum sed, ' +
    'commodo vitae, ornare sit amet, wisi.',
}
				
var helper = $.tm({
  steps = []
});
